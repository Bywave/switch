Main

An element, that will hold all the content in the body. This will help you organize all your content and determine what the content is.



Basic Setup

Use a class name ".main" and use this as a container of all the main content in the body. This is very helpful, when you needed a style like (background color, padding, margins, etc) of all the body content. You can easily style it accordingly.

HTML example markup
<div class="main">
	...
</div>


Main Top

A very simple highlighted content that located in the beginning of the body content. Sometimes used as an introduction and documentation about the site.
